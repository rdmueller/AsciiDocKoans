=== Listen

Listen gehören zu den einfachen Formatierungen.
https://docs.asciidoctor.org/asciidoc/latest/syntax-quick-reference/#lists[📖, role=docs, target=docs]

'''
//solution
* Milch
* Käse
* Eier

'''
//hint
Jeder Listenpunkt beginnt mit `*` gefolgt von einem Leerzeichen.

'''
// Deine Eingabe
Milch
Käse
Eier

''''
Verschachtelte Listen verwenden mehr Sternchen für tiefere Ebenen.

'''
//solution
* Einkaufen:
** Milch
** Käse
** Eier

'''
//hint
`**` für die zweite Ebene, `***` für die dritte.

'''
Einkaufen:
Milch
Käse
Eier

''''
Nummerierte Listen verwenden einen Punkt statt eines Sternchens.

'''
//solution
. Milch
. Käse
. Eier

'''
//hint
`.` statt `*` — die Nummerierung erfolgt automatisch.

'''
Milch
Käse
Eier

''''
Checklisten kombinieren ungeordnete Listen mit Checkboxen.

'''
//solution
* [ ] Milch
* [ ] Käse
* [ ] Eier

'''
//hint
`* [ ]` für eine leere Checkbox, `* [x]` für eine angekreuzte.

'''
Milch
Käse
Eier
